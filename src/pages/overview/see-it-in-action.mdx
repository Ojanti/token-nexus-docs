# See It In Action

The best way to understand TokenNexus is to watch a file get wired up from scratch. The videos below walk through the three phases of the workflow - connecting a repository, mapping variables, and running your first sync.

<VideoPlaceholder
  label="Video: Complete TokenNexus workflow - repo connection to first sync"
  caption="This walkthrough covers a realistic setup with 40 variables, auto-match running to ~80% coverage, and a sync that applies 12 changes."

## Connecting to GitHub (2 minutes)

<VideoPlaceholder
  label="Video: GitHub connection and PAT setup"
  caption="How to generate a Personal Access Token, connect your repository, and select your token files."
/>

The connection process takes about 90 seconds. You need a repository URL, a GitHub PAT, and the folder path where your token files live.

## How mapping works (3 minutes)

<VideoPlaceholder
  label="Video: Mapping variables using auto-match and manual review"
  caption="Auto-match suggestions, how to read confidence indicators, and how to manually map variables the algorithm missed."
/>

Auto-match handles most variables in large files. For anything it misses or gets wrong, the manual dropdown is one click away.
